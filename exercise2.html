Exercise 2

Artist Statement

This is what I imagine it feels like to use Windows XP. You can spawn in windows and drag them around. It's not a very good simulation but it has Bonzi Buddy.

How It Works

I keep track of what images are drawn to the screen using an array. I keep another array to track what hasn't been drawn to the screen. When the spacebar is pressed, I pick an image that hasn't been drawn and put it in the array of images to draw to the screen. Pressing backspace removes images from the top of the array.

To enable image trailing, I simply remove the background() function. To enable the RGB shifting, I use the noise() function to fluctuate between 0 and 255 for the red, green, and blue value.
