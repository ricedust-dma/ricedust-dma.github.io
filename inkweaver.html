Inkweaver

Artist Statement

Inkweaver is a generative art tool for creating spontaneous line art. By adjusting the brush properties, you can create anything from methodical, textured doodles to freeflowing, sporadic lineography. You can find examples here.

How It Works

Lines are drawn by particles that leave a trail as they move across the screen. As the mouse is dragged, more particles are added to the canvas. Each particle has a position, velocity, and acceleration. In order to push particles around, the canvas applies a force to the particle. The angle of this force is determined by the coordinate of the particle mapped to 2D Perlin noise.

Final Thoughts

I think I was able to achieve a pretty distinctive visual style with Inkweaver, and I'm very happy with the possibilities that the different line settings offer. In Art 74, I wrote a drawing program called Lineweaver that explores the beauty of order and symmetry. In contrast, Inkweaver embraces the unpredictable, organic forms that arise from chaos and chance.

I thought about adding colors to the lines and shifting the hue depending on time, position, or noise. In the end, I chose grayscale to maintain simplicity and draw more attention to form and texture. That said, I think that color could add a limitless number of exciting aesthetic possibilities with some more time and thought.

How to Use

Use the mouse to draw on the canvas. Create up to 27 unique brushes by combining multiple line settings.

Hotkey Function
X Clear canvas and reset settings
1, 2, 3 Set line speed
Q, W, E Set line smoothness
A, S, D Set line opacity
Arrow Keys Set flow direction
Z Toggle whether lines wrap around the canvas
F Toggle line fading
C Toggle line erasing
R Reset Settings
